# Pre-redesign archive

These files are from the original iteration of The Wrecking Crew, when the catalog was organized around **named individuals** (Steve Jobs, Codie Sanchez, etc.).

The current design uses **archetypes** — schools of thought exemplified by 2-5 real people whose philosophies genuinely align. The pivot happened because named-individual personas don't compose well: asking for "a finance expert" is too coarse when Buffett and Burry think in fundamentally incompatible ways, and the catalog of named people grows too large to navigate.

Kept here for reference only. Nothing in this folder is loaded by `/crew` or `/crew-review`.

Details of the redesign live in `../DESIGN.md` at the repo root and the `design/` folder.
